
# Software setup
```
Create twitter_auth.json with key from https://developer.twitter.com/en/apps/17453537
{
    "consumer_key": "",
    "consumer_secret": "",
    "access_token": "",
    "access_token_secret": ""
}
sudo pip3 install tweepy / sudo pip install tweepy
Run python tweet_bear.py
Push button(17)
check https://twitter.com/home uglydoll

```
